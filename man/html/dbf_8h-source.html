00185 /* Memo File Structure (.FPT)
00186  * Memo files contain one header record and any number of block structures.
00187  * The header record contains a pointer to the next free block and the size
00188  * of the block in bytes. The size is determined by the SET BLOCKSIZE command
00189  * when the file is created. The header record starts at file position zero and
00190  * occupies 512 bytes. The SET BLOCKSIZE TO 0 command sets the block size width to 1.
00191  */
00192 /* Following the header record are the blocks that contain a block header and
00193  * the text of the memo. The table file contains block numbers that are used to
00194  * reference the memo blocks. The position of the block in the memo file is
00195  * determined by multiplying the block number by the block size (found in the
00196  * memo file header record). All memo blocks start at even block boundary
00197  * addresses. A memo block can occupy more than one consecutive block.
00198  */
00199
